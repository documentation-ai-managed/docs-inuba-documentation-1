## Overview

Inuba Documentation provides a centralized platform where you create, organize, and manage documentation spaces for your projects. You build comprehensive guides, API references, and tutorials using an intuitive MDX-based editor. This space scales with your needs, supporting teams and complex documentation workflows.

<Callout kind="success">
  Start creating your first documentation space in under 5 minutes.
</Callout>

## What is Inuba Documentation

Inuba Documentation is your default documentation hub. You organize content into spaces, each dedicated to a project or product. Features include real-time collaboration, version control, search, and custom theming with colors like `#3B82F6`.

You access powerful MDX components to enrich pages, such as interactive tabs, steps, and code examples. This ensures your documentation remains engaging and easy to maintain.

## Quick Start

Follow these steps to set up your first documentation space.

<Steps>
  <Step title="Create a Space" icon="plus">
    Log in to your Inuba account and click "New Space". Enter a name like "My Project Docs".
  </Step>
  <Step title="Add Content" icon="edit">
    Create a new page using the MDX editor. Start with frontmatter for metadata.
  </Step>
  <Step title="Publish" icon="globe">
    Preview changes, then publish. Share the public URL with your team.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
```javascript
// Example: Fetch space data
const response = await fetch('https://api.inuba.com/spaces/{spaceId}', {
  headers: { Authorization: `Bearer {API_KEY}` }
});
const spaces = await response.json();
```
```python
# Example: Fetch space data
import requests
response = requests.get(
  'https://api.inuba.com/spaces/{spaceId}',
  headers={'Authorization': 'Bearer {API_KEY}'}
)
spaces = response.json()
```
</CodeGroup>
